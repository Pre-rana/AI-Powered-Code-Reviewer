**AI-Powered Code Reviewer**

A simple and efficient code review tool built using Express, React.js and Node.js ERN Stack.
This application allows users to paste or upload code and instantly receive a clear, structured review generated by an AI model. The feedback includes issues detected, optimized fixes and explanations, helping developers write cleaner, safer, and more maintainable code.
This project aims to make code reviews faster, consistent and accessible—especially for beginners who want immediate guidance and cleaner code.

**Features**

1. Instant code analysis

2. Structured feedback: Issues → Fix → Explanation

3. Clean, minimal UI

4. Secure backend API

5. Easy to extend for future features, extensible architecture.

**Tech Stacks**

Frontend: _React (Vite)_

Backend: _Node.js + Express_

AI Model: _Gemini API 2.0_

**Future Scopes**

1. Version Control Integration : Integration of VS Code and GitHub directly to the interface.
   
2. Incorporation of MongoDB : For the transition of the project into professional, multi-user service and as per architectural enhancements.

3. Advanced AI Context and Intelligence : Adding latest models of LLMs for more accurate results.
